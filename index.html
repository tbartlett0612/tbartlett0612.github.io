Projects

This project was done during my schooling for a class on mobile app development. I had chosen to develop an app that allowed the user to keep track of upcoming events. The app has a login/registration screen that utilizes an SQLite DB to store user name and password information. It also took advantage of this DB to link tables of the users to their specific events upon login. This app was built using Android Studio. Here is my Github repository for this project.

This project was done during my Computer Graphics and Visualization course which focused on using OpenGL to recreate a 2D picture into a 3D model. Here is my Github repository for this project.

This project was done during my Embedded Systems class. The purpose of this project was to create a prototype of a smart thermostat using the TI board. For the prototype, it used the TMP006 temperature sensor to read the room temperature (via I2C), an LED to indicate the output to the thermostat where LED on = heat on (via GPIO), two buttons to increase and decrease the set temperature (via GPIO interrupt), and the UART to simulate the data being sent to the server. Here is my Github repository for this project.
